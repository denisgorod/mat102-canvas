Sets are the boxes of mathematics. When you want to reason about any collection of things — numbers, people, colors, functions — you organize them into a set. A set is simply a container with well-defined boundaries: you can always say with certainty whether something is inside the box or outside it.

>[!d] Definition
> A **set** $S$ is any unordered collection of well-defined and distinct objects. An object of a set is called an **element** of that set. The set which contains no elements is called the **empty set** and is denoted $\emptyset$.

"Well-defined" means there is no ambiguity as to whether an element belongs to a set or not. You cannot define "the set of positive numbers that make me happy" — that's not well-defined, because different people (or even the same person on different days) might have different answers. But "the set of positive integers less than 10" is perfectly well-defined: either a number qualifies or it doesn't.

"Distinct" means each element appears at most once. If you try to list the same object twice, you're still describing the same set. Sets also have no natural order — there's no "first" or "last" element unless you impose structure yourself.

>[!idea]
> A set is a collection with a clear boundary: you can always decide whether something belongs or doesn't. The empty set $\emptyset$ is a set with nothing in it.
