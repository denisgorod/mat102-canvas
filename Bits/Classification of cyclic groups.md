You have studied cyclic groups in many forms: $\mathbb{Z}$, $\mathbb{Z}_n$, subgroups generated by a single element. Now you can classify them completely.

>[!s]
> Every cyclic group is isomorphic to exactly one of the following:
> - $\mathbb{Z}$ if the group is infinite
> - $\mathbb{Z}_n$ if the group has finite order $n$

This is a classification theorem: it partitions all cyclic groups into two families, one for each cardinality.

**Proof (infinite case):**

Let $G$ be a cyclic group of infinite order, generated by $g$. Define $\varphi: \mathbb{Z} \to G$ by $\varphi(k) = g^k$.

- **Homomorphism:** $\varphi(k + \ell) = g^{k+\ell} = g^k \cdot g^\ell = \varphi(k) \varphi(\ell)$.

- **Surjective:** Every element of $G$ has the form $g^k$ for some integer $k$, so $\varphi$ hits all of $G$.

- **Injective:** If $\varphi(k) = \varphi(\ell)$, then $g^k = g^\ell$, so $g^{k-\ell} = e$. Since $g$ has infinite order, $k - \ell = 0$, so $k = \ell$.

Thus $\varphi$ is a bijective homomorphism, an isomorphism. $\square$

**Finite case:** For a cyclic group $G$ of order $n$ generated by $g$, the same construction with $\varphi(k) = g^k$ defines an isomorphism $\mathbb{Z}_n \to G$ (details are similar, using the fact that $g^n = e$).

**What this means:** There is *one* infinite cyclic group up to isomorphism (the integers under addition), and for each positive integer $n$, there is *one* cyclic group of order $n$ up to isomorphism (namely, $\mathbb{Z}_n$).

So when you encounter a cyclic group $\langle g \rangle$ of order 5, you know immediately it's isomorphic to $\mathbb{Z}_5 = \{[0], [1], [2], [3], [4]\}$. No matter what $g$ is, no matter what the operation, the structure is always the same.

This vastly simplifies group theory: cyclic groups are completely understood. You need only understand $\mathbb{Z}$ and the family $\{\mathbb{Z}_n : n \in \mathbb{Z}^+\}$.

>[!idea]
> Every cyclic group is isomorphic to $\mathbb{Z}$ or $\mathbb{Z}_n$ — there is one cyclic group structure for each cardinality.
