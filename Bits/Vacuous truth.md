When the hypothesis $P$ is false, $P \Longrightarrow Q$ is automatically true regardless of $Q$. This is called a **vacuous truth** — the implication holds "vacuously" because the hypothesis is never satisfied, so the implication is never tested. "If pigs can fly, then the sky is black" is a true statement. This may feel contrived, but vacuous truths appear frequently in mathematics. For instance, "for all $x \in \emptyset$, $P(x)$" is always true — there are no elements to check, so the statement cannot fail.

>[!idea]
> When the hypothesis is false, the implication is true regardless of the conclusion — this is called a vacuous truth.
