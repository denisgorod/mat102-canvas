Show that $4 \mid (n^2 - 1)$ for any odd integer $n$.

You may use case analysis via the Division Algorithm if it helps, but see if you can find a more elegant approach by factoring or by thinking about what it means for $n$ to be odd.

Try writing $n = 2k + 1$ for some integer $k$, computing $n^2 - 1$, and observing the structure of the result.

>[!question] Exercise
When $n$ is odd, what form does $n^2 - 1$ always take?
