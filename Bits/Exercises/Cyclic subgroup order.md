An element $g$ in a group has an order: the smallest positive integer $n$ such that $g^n = e$. The cyclic subgroup $\langle g \rangle$ generated by $g$ is the set $\{e, g, g^2, \ldots, g^{n-1}\}$.

Here's an obvious question: how many elements are in $\langle g \rangle$? Is the size of the subgroup equal to the order of the element?

Let's check with a concrete example. In $(\mathbb{Z}_{12}, +)$, what is the order of 3? Generate the cyclic subgroup $\langle 3 \rangle$ by repeatedly adding 3: $0, 3, 6, 9, 12 \equiv 0 \pmod{12}$. How many elements did you get before returning to 0?

Now prove it in general: if $g$ has order $n$, show that $|\langle g \rangle| = n$. The key is to verify that $e, g, g^2, \ldots, g^{n-1}$ are all distinct, and that $g^n$ equals the identity.

>[!question] Exercise
> If $g \in G$ has order $n$, show that $|\langle g \rangle| = |g| = n$.
