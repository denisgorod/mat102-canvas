Set-builder notation describes a set by a property. You should be able to go backwards — list the elements explicitly.

>[!question] Exercise
> The following set $S$, written in set-builder notation, is finite. Explicitly write out the elements of $S$.
> $$S = \{z : z = x + y,\; x \in \{1, 2\} \text{ and } y \in \{5, 7\}\}$$
