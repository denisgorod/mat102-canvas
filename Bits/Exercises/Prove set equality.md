Use the double-inclusion technique to prove that two sets defined by different formulas are actually the same.

>[!question] Exercise
> Consider the sets
> $$A = \{n \in \mathbb{Z}^* : n = 4k + 1 \text{ for some } k \in \mathbb{Z}\}$$
> $$B = \{n \in \mathbb{Z}^* : n = 4k - 3 \text{ for some } k \in \mathbb{Z}\}.$$
> Show that $A = B$.
