In an abelian group, many sets turn out to be subgroups. Here's a natural example: let $H = \{h \in G : h^2 = e\}$ be the set of all elements whose square is the identity.

In $(\mathbb{Z}_5^*, \times)$, which elements satisfy $a^2 \equiv 1 \pmod{5}$? (Check $a = 1, 2, 3, 4$.)

Now, assume $G$ is an abelian group and consider $H = \{h \in G : h^2 = e\}$. Show that $H$ is a subgroup using the subgroup test. (You'll need to verify closure and inverses exist.)

Pay special attention to where you use the fact that $G$ is abelian. Does the result hold in nonabelian groups?

>[!question] Exercise
> Let $G$ be an abelian group and $H = \{h \in G : h^2 = e\}$. Show that $H \leq G$ using the subgroup test. Where does abelianness matter?
