Order properties of functions can give us strong conclusions about injectivity.

>[!question] Exercise
> A function $f: \mathbb{R} \rightarrow \mathbb{R}$ is said to be **strictly increasing** if whenever $x < y$, we have $f(x) < f(y)$.
>
> Show that every strictly increasing function is injective.
