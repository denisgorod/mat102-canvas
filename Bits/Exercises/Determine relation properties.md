>[!question] Exercise
> For each relation below, determine which of the following properties are satisfied: reflexive, irreflexive, symmetric, anti-symmetric, transitive, total.
> 1. On $\mathbb{Z}$: $a \sim b$ if $|a - b|$ is even.
> 2. On $\mathbb{Q}$: $a \sim b$ if $ab \in \mathbb{Z}$.
> 3. On $\mathbb{Z}^+$: $a \sim b$ if $1/a < 1/b$.
> 4. On $\mathbb{Z}$: $a \sim b$ if the number of digits in $a$ is less than the number of digits in $b$.
> 5. On $\mathbb{Z} \times \mathbb{Z}$: $(a,b) \sim (c,d)$ if $a + b \leq c + d$.
> 6. On $\mathbb{R}$: $a \sim b$ if $a - b \in \mathbb{Q}$.
> 7. On $\mathbb{R}$: $a \sim b$ if $|a| = |b|$.
