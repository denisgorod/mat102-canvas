Let $\cong$ be an equivalence relation on a set $A$.

>[!question] Exercise
> Show that:
> 1. For any $a \in A$, $a \in [a]$.
> 2. If $a, b \in A$, then $[a] = [b]$ if and only if $a \cong b$.
> 3. If $a, b \in A$, then $[a] \cap [b] = \emptyset$ if and only if $a \not\cong b$.
