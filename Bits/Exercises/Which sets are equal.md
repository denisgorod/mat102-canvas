Remember that sets ignore order and duplicates — but a set of sets is different from the "flattened" set of all those elements.

>[!question] Exercise
> Define the sets
> $$A = \{1, 2, 2, 3, 3, 3\}, \quad B = \{3, 1, 2\}, \quad C = \{\{1, 2\}, \{3\}\}.$$
> Determine which, if any, of these sets are the same.
