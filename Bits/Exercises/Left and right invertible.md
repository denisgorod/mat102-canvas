A function $f: A \to B$ is *left-invertible* if there exists a function $g: B \to A$ such that $g \circ f = \mathrm{id}_A$.

A function $f: A \to B$ is *right-invertible* if there exists a function $g: B \to A$ such that $f \circ g = \mathrm{id}_B$.

(a) Prove that $f$ is left-invertible if and only if $f$ is injective.

(b) Prove that $f$ is right-invertible if and only if $f$ is surjective.

>[!question] Exercise
>In each direction of both proofs, clearly state what you assume and what you must show. Use the definitions of injectivity and surjectivity directly.
