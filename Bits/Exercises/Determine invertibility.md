For each function below, determine whether it is invertible. If so, find an explicit inverse. If not, prove that no inverse exists.

1. $r: \mathbb{R} \to \{0, 1\}$ defined by $r(x) = \begin{cases} 1 & \text{if } x \in \mathbb{Q} \\ 0 & \text{if } x \notin \mathbb{Q} \end{cases}$ (the rational indicator function).

2. $c: \mathbb{R} \to \mathbb{R}$ defined by $c(x) = x^3$.

3. $m: \mathbb{Q} \setminus \{-1\} \to \mathbb{Q} \setminus \{1\}$ defined by $m(x) = \frac{x - 1}{x + 1}$.

4. $g: \mathbb{Z} \to \mathbb{Z}$ defined by $g(n) = \begin{cases} n/2 & \text{if } n \text{ is even} \\ (n - 1)/2 & \text{if } n \text{ is odd} \end{cases}$.

5. $s: \mathbb{Q} \to \mathbb{Z}^+$ defined by $s(a/b) = |a| + |b|$, where $a/b$ is in lowest terms and $b > 0$.

>[!question] Exercise
>Use the bijectivity criterion to determine invertibility. Clearly identify which property (if any) fails for each function.
