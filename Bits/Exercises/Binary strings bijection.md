Infinite binary strings are sequences of 0's and 1's that never end. Can we interpret them in another way?

>[!question] Exercise
> Let $B$ be the set of all infinite binary strings (e.g., $0110100010111\cdots$). Let $N = \{f : \mathbb{Z}^+ \rightarrow \{0,1\}\}$ be the set of all functions from the positive integers to $\{0,1\}$.
>
> Show that there is a bijection between $B$ and $N$.
