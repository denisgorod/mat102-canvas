This exercise combines predicates with set operations.

Let $A = \{-2, -1, 0, 1, 2\}$. Define the following predicates on subsets of $A$:

$P(S)$: $S$ contains at most three elements.
$Q(S)$: $S$ contains exactly one negative number.
$R(S)$: The sum of the elements in $S$ is non-negative.

>[!question] Exercise
> Explicitly write out all of the subsets $S$ of $A$ which satisfy $P(S) \wedge Q(S) \wedge (\neg R(S))$.
