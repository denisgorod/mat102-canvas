Does the image function respect the set operations we know? Let's investigate intersection first.

>[!question] Exercise
> Let $f: A \rightarrow B$ and let $U_1, U_2 \subseteq A$.
> 1. Show that $f(U_1 \cap U_2) \subseteq f(U_1) \cap f(U_2)$.
> 2. Give an explicit counterexample showing that equality does not hold in general; that is, find a function $f$ and sets $U_1, U_2$ such that $f(U_1 \cap U_2) \neq f(U_1) \cap f(U_2)$.
