For each function below, determine whether it is invertible. If so, find an explicit inverse function. If not, prove that no inverse exists.

1. $f: \mathbb{R}^2 \to \mathbb{R}^2$ defined by $f(x, y) = (2x, x - y)$.

2. $g: \mathbb{Z}^+ \to \mathbb{Z}^+$ defined by $g(n) = 10^{n+1} + 1$ (the number with the decimal form $1\underbrace{00\cdots0}_{n \text{ times}}1$).

3. $h: \mathbb{Z}^+ \to \mathbb{Z}^+$ defined by $h(n)$ equals the smallest prime number $p$ such that $n \leq p$.

>[!question] Exercise
>For each function, state whether an inverse exists. If yes, prove that your candidate inverse actually works. If no, identify which property (injectivity or surjectivity) fails and why.
