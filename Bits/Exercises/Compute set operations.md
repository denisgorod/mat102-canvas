Let $T = \{\ldots, -6, -4, -2, 0, 2, 4, 6, \ldots\}$ be the set of all even integers, and let $S = \{0, 1, 2, 3, 4, \ldots\}$ be the set of non-negative integers.

>[!question] Exercise
> Compute $S \cap T$, $S \setminus T$, and $T \setminus S$. Express each answer in set-builder notation.
