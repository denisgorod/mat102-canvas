In an abelian group, the order of a product doesn't matter, so the inverse should satisfy $(ab)^{-1} = a^{-1}b^{-1}$. In a nonabelian group, the formula is different: $(ab)^{-1} = b^{-1}a^{-1}$.

But does this relationship reverse? That is, if $(ab)^{-1} = a^{-1}b^{-1}$ for *all* $a, b \in G$, must $G$ be abelian?

Prove both directions:

**Forward:** If $G$ is abelian, show $(ab)^{-1} = a^{-1}b^{-1}$.

**Backward:** If $(ab)^{-1} = a^{-1}b^{-1}$ for all $a, b \in G$, show that $ab = ba$ for all $a, b \in G$. (Rearrange the equation and use the general inverse-of-product formula.)

>[!question] Exercise
> Show that $G$ is abelian if and only if $(ab)^{-1} = a^{-1}b^{-1}$ for all $a, b \in G$.
