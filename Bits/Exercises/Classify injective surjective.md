A function is injective if distinct inputs map to distinct outputs. It is surjective if every element of the codomain is the image of some element in the domain.

>[!question] Exercise
> For each function below, determine whether it is injective, surjective, both (bijective), or neither. Justify your answer.
> 1. $f: \mathbb{R} \rightarrow \mathbb{R}, \quad x \mapsto \sin(x)$
> 2. $d: \mathbb{R} \rightarrow \mathbb{R}^2, \quad x \mapsto (x, x)$ (the diagonal map)
> 3. $p: \mathbb{R}^2 \rightarrow \mathbb{R}, \quad (x,y) \mapsto x$ (projection)
> 4. $\text{ev}_0 : \text{Poly}_{\mathbb{R}} \rightarrow \mathbb{R}, \quad p(x) \mapsto p(0)$ (evaluation at 0)
