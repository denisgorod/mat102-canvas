For each of the following logical statements, produce a plain English translation. Avoid variable names and negation as much as possible.

1. $\exists x \in \mathbb{Z}^+,\, \exists y \in \mathbb{Z}^+,\, x^2 - 2y^2 = 3$

2. $\forall x \in \mathbb{Z},\, \exists y, z \in \mathbb{Z},\, x = yz$

3. $\forall x \in \mathbb{R},\, \forall y \in \mathbb{R},\, xy < 0 \Longrightarrow (x < 0 \vee y < 0)$

>[!question] Exercise
> Translate each statement above into plain English.
