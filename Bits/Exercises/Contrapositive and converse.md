Write the contrapositive and converse of each of the following (not necessarily true) statements:

1. If $x^2 + y^2 = 4$, then $-2 \leq x \leq 2$.

2. If the product of two real numbers is rational, then both of its factors must be rational.

3. An integer which is divisible by 2 cannot be prime.

4. I will pass this course if I receive a good mark on Term Test 1.

>[!question] Exercise
> For each statement, write both the contrapositive and the converse in clear English.
