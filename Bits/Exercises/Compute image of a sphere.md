The image of a set $U \subseteq A$ under $f: A \rightarrow B$ is
$$f(U) = \{f(x) : x \in U\}.$$

Projection functions "squash" higher-dimensional objects into lower dimensions.

>[!question] Exercise
> Let $p: \mathbb{R}^3 \rightarrow \mathbb{R}^2$ be the projection map $p(x,y,z) = (x,y)$. If $S = \{(x,y,z) : x^2 + y^2 + z^2 = 1\}$ is the unit sphere in $\mathbb{R}^3$, find $p(S)$.
