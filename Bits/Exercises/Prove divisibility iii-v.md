Prove the following three properties. Assume $a, b, c \in \mathbb{Z}$.

**(iii) Subtraction:** If $a \mid b$ and $a \mid (b+c)$, then $a \mid c$.

**(iv) Antisymmetry:** If $a \mid b$ and $b \mid a$, then $a = \pm b$.

**(v) Size bound:** If $b$ is nonzero and $a \mid b$, then $|a| \leq |b|$.

Note: Properties (i) and (ii) were proved in lecture; you now fill in the remaining three.

>[!question] Exercise
Can you prove each of these using only the definition of divisibility and basic properties you already know?
