The $\operatorname{sgn}$ function assigns to each real number its sign:
$$\operatorname{sgn}(x) = \begin{cases} 1 & x > 0 \\ 0 & x = 0 \\ -1 & x < 0 \end{cases}$$

Define a relation on $\mathbb{R}$ by $a \sim b$ if $\operatorname{sgn}(a) = \operatorname{sgn}(b)$.

>[!question] Exercise
> 1. Show that $\sim$ is an equivalence relation.
> 2. Into how many equivalence classes does $\sim$ partition $\mathbb{R}$?
