Now that you can read set-builder notation, let's practice writing sets out explicitly. When you convert from set-builder form to explicit form, you are asking yourself: "What are all the objects that satisfy this condition?"

>[!question] Exercise
> Explicitly write down each of the following sets:
> 1. $\{x \in \mathbb{Z}^+ : 3 < x^2 \leq 80\}$.
> 2. If $S = \{1, 2, 3\}$, then $\mathcal{P}(S) = \{A : A \subseteq S\}$.
> 3. If $A = \{-1, 1\}$ and $B = \{0, 2, 4\}$, then $\{z \in \mathbb{Z}^* : z = y - 2x,\; x \in A \text{ and } y \in B\}$.
> 4. $\{(x, x^2) : x \in \{-1, 0, 1\}\}$.
