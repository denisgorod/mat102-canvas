Create the truth tables for each of the following logical statements:

1. $P \Longleftrightarrow Q$

2. $P \Longrightarrow (Q \vee \neg P)$

3. $P \Longrightarrow (Q \Longleftrightarrow \neg R)$

>[!question] Exercise
> Build truth tables with rows for all possible truth value assignments. Check which assignments make each statement true.
