A set $S \subseteq \mathbb{Z}^+$ is **divisor-closed** if for every element $x \in S$, every divisor of $x$ is also in $S$. For example, $\{1, 2, 3, 4, 6, 12\}$ is divisor-closed. Let $U = \{1, 2, 3, 4, 5, 6, 12\}$.

>[!question] Exercise
> 1. Let $D$ denote the set of divisor-closed subsets of $U$. Write out all the elements of $D$.
> 2. Define a partial order on $D$ by inclusion: $A \preceq B$ if $A \subseteq B$.
> 3. What are the maximal and minimal elements of $D$?
> 4. Let $S = \{\{1,2\}, \{1,3\}\}$. Find all the upper bounds for $S$ in $D$. What is $\sup(S)$?
> 5. Find all the lower bounds for $S$. What is $\inf(S)$?
