# p|n iff p|n²

**Show that if $p$ is prime and $n \in \mathbb{Z}$, then $p \mid n$ if and only if $p \mid n^2$.**

This is a useful application of the prime divisibility property. Establish both directions: first show that $p \mid n$ implies $p \mid n^2$ (this is straightforward), then show the converse.

For the converse, suppose $p \mid n^2 = n \cdot n$. Use the prime divisibility property: if $p$ divides the product $n \cdot n$, what can you conclude?

>[!question] Exercise
>The key is the prime divisibility property: if $p \mid n \cdot n$ then $p \mid n$ or $p \mid n$.
