The integers $(\mathbb{Z}, +)$ are a fundamental example of a cyclic group. But what is the generator?

Is $\mathbb{Z}$ cyclic under addition? If so, what element generates it? Can you generate all of $\mathbb{Z}$ by repeatedly adding 1? What about 2?

Now, describe the cyclic subgroup $\langle 2 \rangle$ generated by 2. This is the set of all integers you can write as $2k$ for $k \in \mathbb{Z}$. It's a familiar set — what is it?

More generally, for any nonzero integer $n$, describe $\langle n \rangle$. Is it the same as $\langle -n \rangle$?

>[!question] Exercise
> Is $\mathbb{Z}$ cyclic under addition? If so, find a generator. Then describe the cyclic subgroup $\langle n \rangle$ for any nonzero integer $n$.
