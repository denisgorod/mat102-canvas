Prove each of the following statements about cardinality:

(a) The set of irrational numbers $\mathbb{R} \setminus \mathbb{Q}$ has the same cardinality as $\mathbb{R}$.

(b) Let $F = \{A \subseteq \mathbb{Z}^+ : A \text{ is finite}\}$. Show that $F$ is countable.

(c) The set of all polynomials with rational coefficients, denoted $\mathbb{Q}[x]$, is countable.

(d) Show that the set $\{f : \mathbb{Z}^+ \to \{0, 1\}\}$ of all binary-valued functions on the positive integers has the same cardinality as $\mathbb{R}$.

>[!question] Exercise
>For parts (a)–(c), you may use facts from previous work (e.g., the countability of $\mathbb{Q}$, the cardinality of intervals). For part (d), recall that binary strings can be viewed as elements of $\{0, 1\}^{\mathbb{Z}^+}$ and relate this to the Cantor–Schröder–Bernstein theorem or diagonalization.
