Use the Euclidean Algorithm to compute $\gcd(a, b)$ for each of the following pairs:

(a) $a = 252, b = 198$

(b) $a = 385, b = 595$

(c) $a = 420, b = 378$

(d) $a = 4620, b = 6930$

For each pair, show the sequence of divisions used in the Euclidean Algorithm. Verify your answer by checking that your claimed gcd divides both $a$ and $b$.

>[!question] Exercise
Does the Euclidean Algorithm always terminate, and does the last nonzero remainder always give the gcd?
