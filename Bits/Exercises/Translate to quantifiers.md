>[!question] Exercise
> Write each of the following sentences using predicates and appropriate quantifiers ($\forall$, $\exists$):
> 1. Every cow has a favourite radio station.
> 2. There is a black horse.
> 3. In every sport, there exists someone who breaks the rules.
> 4. There is one textbook that every class uses.
> 5. There is no largest real number.
