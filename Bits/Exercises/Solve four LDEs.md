# Solve four LDEs

For each linear Diophantine equation below, find:
**(i)** a single integer solution,
**(ii)** the general solution (all integer solutions),
**(iii)** all non-negative integer solutions, if they exist.

**(a)** $221x + 143y = 13$

**(b)** $286x + 121y = 33$

**(c)** $1700x + 867y = 17$

**(d)** $1232x + 553y = 63$

For part (iii) on each equation, you will need to find which values of $n$ make both $x$ and $y$ non-negative, and count how many integer values of $n$ satisfy those constraints.

>[!question] Exercise
>Use the extended Euclidean algorithm for each coefficient pair, check divisibility, find one solution, apply the general formula, then solve the inequality constraints for non-negativity.
