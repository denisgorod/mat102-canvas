Think about what left-totality and functionality mean in the extreme cases.

>[!question] Exercise
> 1. Is there a function $f: \emptyset \rightarrow X$ for any nonempty set $X$? Explain by checking left-totality and functionality.
> 2. Is there a function $g: X \rightarrow \emptyset$ for any nonempty set $X$? Explain by checking left-totality and functionality.
> 3. What about a function from $\emptyset$ to $\emptyset$?
