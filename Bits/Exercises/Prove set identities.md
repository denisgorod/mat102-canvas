Sets obey algebraic laws, just like numbers do. Here you prove several fundamental identities by showing that two sides of an equation contain exactly the same elements. You can do this by element-chasing or by using definitions.

>[!question] Exercise
> Let $U$ be the universe of discourse with $A, B, C \subseteq U$. Prove each of the following set identities:
> 1. $A \cap A^c = \emptyset$
> 2. $A \setminus B = A \cap B^c$
> 3. $A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$
> 4. $A \setminus (B \cup C) = (A \setminus B) \cap (A \setminus C)$
> 5. $(A \cap B) \times C = (A \times C) \cap (B \times C)$
