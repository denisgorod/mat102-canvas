A palindromic number is a number that reads the same forwards and backwards. For example, $12321$ is palindromic, while $10011$ is not.

>[!question] Exercise
> Identify the hypothesis and conclusion of the following statement:
>
> *Any 4-digit palindromic number is divisible by 11.*
