# Prime property in logic

**Write the prime divisibility property using propositional logic notation.**

Recall that a number $p$ is prime if and only if: for any integers $a, b$, if $p \mid ab$ then $p \mid a$ or $p \mid b$.

Introduce symbols like:
- $P(p)$ to mean "$p$ is prime"
- $D(x, y)$ to mean "$x$ divides $y$"
- $A, B, O$ to mean "for all" (∀), "there exist" (∃), and "or" (∨)

Then write the statement "if $p$ is prime, then for all $a, b$, if $p \mid ab$ then $p \mid a$ or $p \mid b$" using formal logic symbols.

>[!question] Exercise
>Unpack "for any $a,b$" and "if...then" into quantifiers and logical operators; use $D(\cdot, \cdot)$ and $\vee$ appropriately.
