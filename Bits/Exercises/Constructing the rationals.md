Define a relation on $\mathbb{Z} \times \mathbb{Z}^+$ by $(a, b) \sim (c, d)$ if $ad = bc$.

>[!question] Exercise
> 1. Show that $\sim$ is an equivalence relation.
> 2. Give three elements of the equivalence class $[(1, 2)]$.
> 3. We can define addition of equivalence classes as $[(a,b)] + [(c,d)] = [(ad + bc,\, bd)]$. Show that this is **well-defined**: the result does not depend on which representative you choose from each class.
> 4. We can define multiplication as $[(a,b)] \times [(c,d)] = [(ac,\, bd)]$. Show that this is well-defined.
> 5. What have we just defined?
