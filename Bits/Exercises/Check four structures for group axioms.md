Now that you know the three group axioms, it's time to test some candidates.

For each of the following structures, determine whether it is a group. If it is, verify all three axioms. If it is not, identify which axiom(s) fail and give a counterexample or explanation.

**(a)** $(\mathbb{Z}^+, \text{exponentiation})$: the positive integers with the operation $a \otimes b = a^b$.

**(b)** $(\mathbb{Z}, \otimes)$ where $a \otimes b = a + b + ab$.

**(c)** $(\mathbb{Q} \setminus \{0\}, \otimes)$ where $a \otimes b = \frac{a}{b}$.

**(d)** Rock-Paper-Scissors, where the "operation" is "two players play and the winner's move is the result." (Be creative about how to model this—what is the set? What is the identity?)

>[!question] Exercise
> For each of the four structures above, determine whether it is a group under the given operation. Verify or refute each of the three axioms.
