In mathematics, certain sets come up so often that they earn their own special symbols. When you see these symbols, they always refer to the same set.

The **integers** are
$$\mathbb{Z} = \{\ldots, -3, -2, -1, 0, 1, 2, 3, \ldots\}.$$
The symbol $\mathbb{Z}$ comes from the German word "Zahl" (number).

The **rational numbers** are
$$\mathbb{Q} = \left\{\frac{a}{b} : a, b \in \mathbb{Z},\, b \neq 0\right\}.$$
These are all numbers you can express as a fraction of integers (with nonzero denominator). The symbol $\mathbb{Q}$ stands for "quotient."

The **real numbers** $\mathbb{R}$ include integers, rationals, and also irrational numbers like $\pi$ and $\sqrt{2}$. For now, think of $\mathbb{R}$ as all numbers you can place on a number line.

One point of confusion: there's no universal agreement on whether the "natural numbers" include 0 or start at 1. To avoid ambiguity, we use explicit notation:
- $\mathbb{Z}^+ = \{1, 2, 3, \ldots\}$ for positive integers (never 0 or negative)
- $\mathbb{Z}^* = \{0, 1, 2, \ldots\}$ for non-negative integers (includes 0, no negatives)
- $\mathbb{Z} = \{\ldots, -1, 0, 1, \ldots\}$ for all integers

When someone writes $\mathbb{N}$, check what the author means — it varies by convention.

>[!idea]
> Standard number sets have symbols: $\mathbb{Z}$ (integers), $\mathbb{Q}$ (rationals), $\mathbb{R}$ (reals). Use $\mathbb{Z}^+$ and $\mathbb{Z}^*$ to avoid confusion about whether 0 and negatives are included.
